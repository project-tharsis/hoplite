## 5. 评估规则摘要

### 模型1: 文化是战术的操作系统
**哲学**: 日常标准不对，战术不会稳定运行。
**正面指标**:
  - 纪律良好：≤1黄牌
  - 黄牌出现在合理时刻（如战术犯规）
  - 犯规数可控
**负面指标**:
  - 红牌
  - 2+黄牌且出现时机不当（开场/僵持时无谓吃牌）
  - 犯规失控
**判断规则**:
  - red > 0 → 🔴 (high confidence)
  - yellow >= 4 → 🔴 (high confidence)
  - critical-time card (yellow >= 2 AND result == L) → 🔴 (high confidence)
  - fouls_delta >= 8 → 🔴 (high confidence)
**叙事指导**: 领先7-0时89分钟的战术黄牌 ≠ 0-0时2分钟的鲁莽黄牌。结合比赛状态判断纪律质量。

### 模型2: 控制比赛发生在哪里
**哲学**: 控制不是只控球，是把比赛带到想要的区域、节奏和情绪。
**正面指标**:
  - 控球≥55%
  - 射门显著多于对手
  - xG碾压
**负面指标**:
  - 控球≤45%
  - 射门少于对手
  - 被锁在防守三区
**判断规则**:
  - shot_delta > 0 AND xg_delta > 0 → 🟢 (high)
  - xg missing but shot_delta > 0 AND possession_delta > 0 → 🟢 (medium)
  - shot_delta < 0 AND xg_delta < 0 AND corner_delta < 0 → 🔴 (high)
  - mixed signals (poss positive but shots negative) → 🟡 (medium)
**叙事指导**: 控球率高不等于控制比赛。需要结合射门质量和区域控制来判断阿森纳是否真的在主导比赛。

### 模型3: 防守也是进攻身份
**哲学**: 防守应为进攻创造平台——球员要'爱'防守。
**正面指标**:
  - 零封
  - 失≤1球且对手射正≤3
  - 防守为进攻提供了平台
**负面指标**:
  - 失3+球
  - 防守崩溃导致进攻被压缩
**判断规则**:
  - clean sheet → 🟢 (high)
  - 1 conceded AND opp_sot <= 3 → 🟢 (high)
  - 3+ conceded → 🔴 (high)
  - xg_against > 2.0 → 🔴 (medium)
**叙事指导**: 防守不只是不失球。要判断防守是否为进攻创造了平台——高位反抢成功后快速反击是好防守转化为进攻的标志。

### 模型4: 边际收益要专家化
**哲学**: 能决定比赛的部门要专家化——定位球/转换不能靠兼职。
**正面指标**:
  - 定位球进球≥2
  - 零定位球失球
  - 角球有威胁
**负面指标**:
  - 定位球失球≥2
  - 定位球进攻毫无威胁
**判断规则**:
  - set_piece_goals_for > set_piece_goals_against → 🟢 (high)
  - even but corner advantage > 3 → 🟢 (high)
  - set_piece_goals_against > set_piece_goals_for → 🔴 (high)
  - neutral → 🟡 (high)
**叙事指导**: 定位球不只是进球数。要评估角球/任意球的威胁质量——即使没进球，如果每次角球都制造混乱，说明定位球教练组工作到位。

### 模型5: 加能力，但不要丢身份
**哲学**: 保留传统优势，同时叠加新能力。
**正面指标**:
  - 传统优势保留（传球≥85%、控球好）
  - 新元素有效（多人进球、新阵型成功）
**负面指标**:
  - 传统优势丢失（传球差、控球被压制）
**判断规则**:
  - win AND pass_accuracy > 80% AND possession > 50% → 🟢 (high)
  - loss → 🔴 (high)
  - draw with poor control (PA<75 AND poss<45) → 🔴 (medium)
  - win but one metric below threshold → 🟡 (medium)
**叙事指导**: 多人进球说明体系运转良好，一人包办说明依赖个人能力。传统优势（控球、传球）不能因为叠加新战术而丢失。

### 模型6: 人需要清晰度，不只是压力
**哲学**: 替补要知道如何贡献，球员要角色清晰。
**正面指标**:
  - 换人产生进球/助攻
  - 替补融入体系
  - 换人时机合理（45-75分钟）
**负面指标**:
  - 换人过晚（80'后且比分落后）
  - 多次换人无效
  - 替补明显脱节
**判断规则**:
  - no subs AND result == L → 🔴 (high)
  - no subs → 🟡 (medium)
  - earliest sub < 70' AND goals_after_subs > 0 → 🟢 (high)
  - latest sub > 80' AND goals conceded after subs → 🔴 (high)
**叙事指导**: 换人不只是'谁上谁下'。要评估替补上场后是否清楚自己的角色——是加强防守还是寻找进球？角色清晰度决定换人效果。
